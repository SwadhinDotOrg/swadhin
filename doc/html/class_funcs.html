setStatusMsg ( $  msg,
$  status = MSGBOX_ERROR 
)

Call this function within Controller classes to set a HTML message which you want to get displayed automatically in the generated HTML for the page. The messages are of various types, i.e Success, Warning, Information or type.
Each type has it's own style, i.e. Success type messages are displayied withing Green background, where Failure type are displayed withing Red background/color styles.

Note:
This function safely starts PHP Session - as status messages are passed using PHP Session.
Parameters:
string$msgThe HTML message you want to get displayed
int$statussee constants defined for parameter $type in messageExit() function
