Detailed Description

Validates user-submitted inputs.

Author:
Shafiul Azam

General purpose validation class. $core has an object of this class named $validate

$core->validate is heavily used in validating web forms (child of class CoreForm )

Most of the functions (except userInput() ) of this class are used form validation.

You can add your own functions in the custom validator class Validator in custom/class folder.

You can perform a series of validation (calling several functions of this class, one after another) on the same string & append error messages to member variable $errorArray (automatically)- if any of these fucntions fail. The string on which these validation operations will take place is first stored in member variable $subject. Then you call one/more functions (automatically called if you specify this function names in the object of CoreForm seperated by "|" - see 3rd parameter of CoreForm::element() function.)
